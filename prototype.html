Prototype - Transmission Line Fault Detection

In the prototype phase, we implement the Random Forest algorithm for predicting transmission line faults based on current and voltage measurements.

Random Forest Algorithm

We use the Random Forest algorithm to classify fault types based on input data. This ensemble learning method combines multiple decision trees to make predictions. It has proven to be effective in fault detection due to its ability to handle complex data relationships.

The algorithm takes current and voltage values as input and outputs one of the following fault types:

    • Ground Fault
    • Fault in Line A
    • Fault in Line B
    • Fault in Line C
    • LG fault (Between Phase A and Ground)
    • LG fault (Between Phase B and Ground)
    • LG fault (Between Phase C and Ground)
    • LL fault (Between Phase B and Phase A)
    • LL fault (Between Phase C and Phase B)
    • LL fault (Between Phase C and Phase A)
    • LLG Fault (Between Phases A, B, and Ground)
    • LLG Fault (Between Phases A, C, and Ground)
    • LLG Fault (Between Phases C, B, and Ground)
    • LLL Fault (Between all three phases)
    • LLLG Fault (Three phase symmetrical fault)

This approach allows us to proactively identify and address transmission line issues, ensuring the reliability of the power distribution system.

The prototype phase is a critical step in demonstrating the effectiveness of our fault detection system. By utilizing the Random Forest algorithm, we aim to provide accurate and timely fault predictions for enhanced power grid management.
